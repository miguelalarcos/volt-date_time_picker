# Volt::DateTimePicker

A date-time picker.

## Installation

Add this line to your application's Gemfile:

    gem 'volt-date_time_picker'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install volt-date_time_picker

## Usage

<:date_time_picker time={{true}} value="{{page._datetime}}" date_format='%d-%m-%Y %H:%M' />
